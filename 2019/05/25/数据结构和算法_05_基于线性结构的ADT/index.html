05_基于线性结构的ADT

发表于 2019-05-25 | 分类于 数据结构和算法

线性表（List）

线性表ADT的定义

ADT 线性表（List）

Data

  • 线性表的数据对象集合为{a1,a2,…,an}，每个元素的类型均为DataType。其中，除第一个元素a1外，每一个元素有且只有一个直接前驱元素，除了最后一个元素an外，每一个元素有且只有一个直接后继元素。数据元素之间的关系是一对一的关系。

Operation

  • 判断List是否为空
  • 获取表中元素的个数
  • 访问第 k 个元素
  • 查找指定元素在表中的位置
  • 在第 k 个位置插入一个新元素
  • 追加一个元素
  • 删除第 k 个元素
  • 复制一个线性表

这里只列举了一些基本的操作，在实际编程过程中，需要根据不同的业务需求选择合适的操作。

栈（Stack）

栈（Stack）是一个后进先出（Last in first out,LIFO）的线性表，它只允许在表的一端（栈顶）进行删除和插入操作。

栈ADT的定义

ADT 栈（Stack）

Data

  • 与线性表相同

Operation

  • 入栈——在栈顶插入一个元素
  • 出栈——在栈顶删除一个元素，并返回

队列

队列（Queue）是一种先进先出（First In First Out, FIFO）的线性表，它只允许在一端进行插入操作，而在另一端进行删除操作。

队列ADT的定义

ADT 队列（Queue）

Data

  • 与线性表相同

Operation

  • 入队——在表的一端插入一个元素
  • 出队——在表的另一端删除一个元素，并返回该元素

双端队列

双端队列（Deque）是一种前端和后端都支持插入和删除操作的队列。

双端队列ADT的定义

ADT 双端队列（Deque）

Data

  • 与线性表相同

Operation

  • 前端插入——在表的前端插入一个元素
  • 前端删除——在表的前端删除一个元素，并返回该元素
  • 后端插入——在表的后端插入一个元素
  • 后端删除——在表的后端删除一个元素，并返回该元素
